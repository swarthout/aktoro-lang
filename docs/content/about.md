## About Othello

[Othello](https://github.com/swarthout/othello) is an open source programming language that makes it easy to write simple, fast, and expressive code while leveraging the powerful Golang ecosytem.

## Features

- Pattern matching
- Type inference
- Familiar Syntax
- Concurrency
